# Connect Bridgeapp for XBOX Elite 2 Gamepad

> Nodejs based program. You must use node which has version 10.15 over. Please check the version of node by `node --version `

# How to run

## Install

```sh
npm install
```

## Run

```sh
npm run start
```

# How to run demo

```sh
npm run demo
```
